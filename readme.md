# ShopSphere Pro 🛒

A modern, responsive E-Commerce Product Page built with HTML, CSS, and JavaScript in a single file.

## Features

- Product Listing Grid
- Product Search
- Category Filter
- Shopping Cart
- Cart Total Calculation
- LocalStorage Cart Persistence
- Dark / Light Mode
- Quick View Product Modal
- Toast Notifications
- Responsive Design
- Modern UI

## Technologies Used

- HTML
- CSS
- JavaScript (Vanilla JS)


## How to Run

1. Download `week_3.md`
2. Open the file in any modern web browser.
3. Browse products, search, filter, and add items to cart.

## Main Functionalities

### Search Products
Use the search bar to find products instantly.

### Category Filter
Filter products by category.

### Shopping Cart
Add or remove products and see the total price update automatically.

### Dark Mode
Switch between light and dark themes.

### Local Storage
Cart data remains saved after page refresh.
